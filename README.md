# FileRenameApp
A WPF application for managing and renaming submission files.

大学の命名規則に従い、選択したファイルの名前を自動で変更するC#で作成したWPFアプリです.

本アプリは、レポート提出時の命名規則の確認や手動入力の手間を減らすことを目的としています。
また、ファイル管理が苦手な方でも簡単に整理できるよう設計しています。
How to Use
1. 起動

ダウンロード後、ZIPを解凍し「Rename」をダブルクリックして起動する。

2. 初期設定

時間割表が表示されたら、右下の「基本設定」から以下を設定する：

学籍番号
名前
学年

設定後、「保存」を押す。

3. 保存先フォルダ

デフォルトの保存先：

Downloads/提出フォルダ

※基本的に変更は推奨しない

4. 共通変数の設定

共通で使用する命名規則（全教科共通部分）を追加できる。

5. ファイルの保存手順
時間割画面に戻る
任意のコマを選択
教科設定画面を開く
教科名を入力（フォルダ整理用として推奨）
命名規則を選択
「ファイル選択」からファイルを選ぶ
「保存実行」を押すと保存先へ出力される

7. オリジナル保存

「オリジナル保存」ボタンは、提出用ではなく以下の用途：

授業資料の整理
提出以外のファイルのリネーム

7. 自動カウンタ変数

以下の変数は保存ごとに +1 される：
回数
オリジナル回数

値を変更したい場合は、該当変数名と変更値を指定して更新できる。
